2. 遊戲規則 (Game Rules)

2.1 遊戲流程

  1. 玩家下前注（Base Bet） → 開啟兩張牌
  2. 玩家選擇是否下第二注（Double/Continue Bet）
  3. 若下注 → 開第三張牌
  4. 判定「射入」「撞柱」或「失敗」並結算
  5. 回合結束 → 進入下一局

2.2 賠率與事件說明

  • 射入 （第三張落在兩張牌之間）：賠率 3X（待討論）
  • 撞柱 （第三張與任一張同點數）：額外賠付下注金額 ( -1X )
  • 失敗 （在外側）：全損
  • 放棄第二注 ：只損前注

2.3 牌值定義

  • A = 1
  • J = 11
  • Q = 12
  • K = 13

2.4 其他規則

  • 撲克牌數量為四副 (降低玩家算機率可能性)
  • 撲克牌每局進行重新洗牌
  • 每一張桌子最多五名玩家同時進行遊戲
